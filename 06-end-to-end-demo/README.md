# End-to-End Demo

This walkthrough demonstrates the full lifecycle of an AI-powered procurement bidding evaluation — from project setup through automated scoring.

---

## Step 1 — Download the Demo Assets

Download and extract the demo package, which includes the bidding prompt templates and sample PDF documents used throughout this tutorial.

[Download demo assets (biddemo.zip)](./assets/biddemo.zip)

---

## Step 2 — Create a Bidding Project

Create a new bidding project by uploading the bidding documents and configuring the evaluation prompt.

![Creating a bidding project with documents and prompt](./img/1CreateBidingProject.gif)

---

## Step 3 — Submit a Bid

Add a bid submission to the project. Each submission represents a vendor response that will be evaluated by the AI model.

![Submitting a bid to the project](./img/2BidSubmission.gif)

---

## Step 4 — Evaluate and Review Results

Trigger the AI evaluation and review the generated scores and recommendations for each bid submission.

![Running the evaluation and reviewing results](./img/3BidEvaluation&CheckResult.gif)
